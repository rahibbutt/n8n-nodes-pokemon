# n8n Custom Node: Pokémon Data

A custom n8n node to fetch Pokémon data from the [PokeAPI](https://pokeapi.co/).  
This node allows you to retrieve detailed information about Pokémon, including stats, types, and abilities.  
It can be used to build fun Pokémon-themed workflows, practice creating custom n8n nodes, or learn API integrations and automation.

---

## Features

- Fetch Pokémon by name or ID.
- Access Pokémon stats, types, and abilities.
- Ideal for learning custom n8n node development and API handling.
- Can be integrated with other nodes to create automated workflows (Slack notifications, Google Sheets logging, email digests, etc.).
